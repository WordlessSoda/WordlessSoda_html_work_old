Microsoft HoloLens

Microsoft HoloLens 是微软首个不受线缆限制的全息计算机设备，能够让你与数字内容交互，并与周围真实环境中的全息影像互动。

Microsoft HoloLens, picture from Microsoft

为什么选择 HoloLens

它所能带给你的

  1. 通过 HoloLens 实现混合现实
  2. 每个人的混合现实
  3. 全息影像增强了现实世界
  4. 一种更自然的交互方式
  5. 连接、创造和探索
  6. 协作和沟通
  7. 创造你想像的世界
  8. 将你的工作可视化
  9. 探索位置和想法
  10. 发现、想像、创造

它将为世界带来变革

  1. 混合现实能将物理世界和数字世界中的人、物和场所融为一体。这个混合环境即是你的画布，你可以在其中创作和享受多元化体验。
  2. Windows 混合现实将虚拟现实的紧张刺激感和参与并个性化打造你的环境的功能融为一体。Windows 10 将为各种头戴设备提供这种体验，从完全不受线缆控制的全息计算机（如 HoloLens）到最具创新性和经济实惠的头戴设备。
  3. 与混合现实中的全息影像交互，让你能够将数字内容可视化，并将其作为现实世界的一部分使用。使用 Microsoft HoloLens 实现混合现实的前景
  4. 全息影像可对你和你所处环境作出反应。Microsoft HoloLens 让你能够以尽可能自然的方式与内容和信息进行交互。
  5. 变革你沟通、创造、协作和探索的方式。当你可以创建和使用与你周围环境相关的全息影像时，你的想法将更有可能成为现实。
  6. 当协同工作时，展示比讲述更清晰。你的 Skype 联系人可以在你视图中的物理对象上叠加草图和全息影像。
  7. 优化全息影像 - 调整设计、分享想法，了解你的创作与现实世界的关系。
  8. 通过在 3D 空间中工作，超越 2D 渲染可以完成的事项。在你能够检查每个有利位置时，就可以更快地做出更明智的决策和更智能的原型。
  9. 从你同事的视角看全息影像，即使她远在世界的另一端。从里到外探索现实世界中的想法。
  10. 体验为混合现实构建的首款应用，从中激发自己的灵感。想像你可以做什么。
